# Restaurant-powerbi-sql
Restaurant Business Intelligence Dashboard using Power BI & SQL;

# Project Overview

This project analyzes restaurant sales, customers, and performance metrics using SQL and Power BI to generate actionable business insights.

# Tools Used

1.SQL (MySQL)
2.Power BI
3.Excel / CSV
4.GitHub

# Key Features

1.Sales Analysis
2.Top Performing Dishes
3.Monthly Revenue Trends
4.Customer Insights
5.Interactive Power BI Dashboard

# Dataset

Restaurant transactional data including orders, customers, and products.
